Problems with current structure
✕ Each calculator has no unique URL
✕ No dedicated landing pages for SEO
✕ No blog section exists
✕ No llms.txt or structured data
✕ Single title/description for entire site
Proposed Structure
Every page gets its own URL + metadata
✓ / — Homepage with calculator overview
✓ /sip-simulator — Dedicated SIP page
✓ /emi-analyzer — Dedicated EMI page
✓ /tax-sandbox — Dedicated Tax page
✓ ... (8 more calculator routes)
✓ /blog — Blog index
✓ /blog/[slug] — Individual posts
✓ /admin — CMS admin panel
★ /llms.txt — AI agent ingestion
★ /llms-full.txt — Full content export
★ /sitemap.xml — Search engine sitemap
