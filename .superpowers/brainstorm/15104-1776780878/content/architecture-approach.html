Key Design Decisions

Why this works

  • No external database — content lives in MDX files + a lightweight JSON config, committed to the repo
  • Anti-marketing intact — still zero PII, zero tracking, all math client-side
  • AI-agent discoverable — llms.txt gives agents a structured map of the entire site
  • Build-time SEO — all metadata generated at build time, no runtime overhead
  • Incremental build — each layer can be deployed independently

Trade-offs

  • Admin requires deploy — content changes need a git push + rebuild (not instant, but Vercel deploys in ~30s)
  • No user accounts — admin panel uses a simple password gate (no auth system, stays zero-PII)
  • SEM is read-only dashboard — we can show metrics and suggestions, but actual ad campaigns need external tools (Google Ads, etc.)
